NEW PRODUCT DIRECTION
---------------------
Product name:
API Contract Diff

Core idea:
A developer tool that compares two API contracts and reports breaking/non-breaking changes clearly.

Supported inputs for V1:
- OpenAPI / Swagger YAML
- OpenAPI / Swagger JSON

Core capabilities:
- Compare old contract vs new contract
- Detect breaking changes
- Detect additive changes
- Produce a clean summary
- Produce detailed diff sections
- Provide severity levels such as HIGH / MEDIUM / LOW
- Support a CLI-first workflow in the local implementation
- The frontend dashboard should be a demo layer around the diff engine

BACKEND / CORE ENGINE

Suggested architecture:
